## ⚙️ Installation

### Clone Repository

git clone https://github.com/amalmk-codes/CodeAlpha_EcommerceStore.git

cd CodeAlpha_EcommerceStore

### Backend Setup

cd server

npm install

Create .env file:

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

PORT=5000

Run backend:

npm run dev

### Frontend Setup

cd client

npm install

npm run dev
